#### Odpalanie
Kompilacja:
```bash
make all
```
odpalenie serwera w jednym terminalu:
 ```bash
 ./server
```

odpalenie różnych klientów w różnych terminalach:
```bash
./client 
```
w kazdym terminalu

wczytywanie z pliku w kliencie: 
```bash
READ scieżka_do_pliku_z_komendami
```

komendy:
```bash
ECHO string

LIST

FRIENDS lista_id_klientów

2ALL string

2FRIENDS string

2ONE id_klienta string

STOP
```
